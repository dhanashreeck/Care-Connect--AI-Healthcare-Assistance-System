# AI-Enabled Healthcare Assistance System

## Overview

The AI-Enabled Healthcare Assistance System is a web-based platform developed as a final-year academic project. The system helps users book doctor appointments, access healthcare-related assistance through a chatbot, and generate simplified prescription summaries.

## Features

* Doctor appointment booking system
* Healthcare chatbot for user support
* Prescription summarization functionality
* Patient information management
* User-friendly web interface

## Technologies Used

* Python
* MySQL
* HTML
* CSS
* JavaScript

## My Contributions

* Implemented prescription summarization features
* Designed and managed the MySQL database
* Developed the frontend interface using HTML, CSS, and JavaScript
